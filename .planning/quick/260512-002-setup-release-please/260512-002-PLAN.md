# Quick Task 260512-002: Remove all current releases and add release-please

## Task 1: Remove existing releases and tags

**Action:** Delete all GitHub releases (v0.1.1 through v0.1.21) and remote/local tags.
**Verify:** `gh release list` returns empty. `git tag | grep v0.1` returns nothing.

## Task 2: Add release-please configuration

**Action:** Create three files based on https://github.com/pau-vega/nordvpn-action reference:
- `.github/release-please-config.json` — release-please config (`release-type: node`, initial version 0.0.1)
- `.release-please-manifest.json` — manifest tracking version `{".": "0.0.1"}`
- `.github/workflows/release-please.yml` — GitHub Actions workflow

**Verify:** All three files exist with correct content.

## Task 3: Update package.json version

**Action:** Set `version` field to `"0.0.1"` in root `package.json`.
**Verify:** `jq '.version' package.json` outputs `"0.0.1"`.
